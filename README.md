# Hangman Word Game

## USAGE

To start the program, run the command "python src/main.py" or "python main.py"

## TESTING

Execute the hangman_testing.py file, through
the command "python src/hangman_testing.py" if positioned at the root, or "python hangman_testing.py" if positioned in src folder.

The expected output is :
Test 1 Passed: Correct guess handled.
Test 2 Passed: Incorrect guess deducted life.
Test 3 Passed: Duplicate guess ignored.
Test 4 Passed: Case sensitivity handled.

All tests passed successfully!
